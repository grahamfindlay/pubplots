# Figure requirements by journal

This document summarizes formatting requirements for figures across various journals.

Note that Science's 2025 author preparation guide ([1](https://www.science.org/cms/asset/6ebc81c2-e38a-4cea-b0c2-a0402817f13e/author_prep_guide_2025.pdf), for a 2-column format) directly contradicts its easier-to-find instructions ([2](https://www.science.org/content/page/instructions-preparing-revised-manuscript)) for the old 3-column format. Ignore the latter.

## Figure typography requirements

| Journal | Family | Fonts | Font size | Source |
|---------|--------|----------------|-------------------------|--------|
| Nature Neuroscience | Nature | Arial, Helvetica | 5–7 pt | [1](https://www.nature.com/documents/NRJs-guide-to-preparing-final-artwork.pdf) |
| Science | AAAS | Arial, Helvetica | Panel parts: 10pt bold; Labels: 6-9pt| [1](https://www.science.org/cms/asset/6ebc81c2-e38a-4cea-b0c2-a0402817f13e/author_prep_guide_2025.pdf) |

### Nearly ubiquitous typography requirements

- All journals require sans-serif fonts for figure text.
- The same typeface should always be used across figures.
- Never use more than 3 discrete font sizes across all figures. For example, Science allows labels to be 6-9 pt, but only 3 discrete sizes within that range (e.g. 6, 7, and 8, but not 9).
- Text must always be editable as text, never rasterized, flattened, outlined, or converted to paths.

### Nearly ubiquitous other requirements

- Always provide figures in RGB color format, never CMYK.
- If something can be provided in a vector graphics format, do it.
